```ts
const invokePromise = supabase.functions.invoke('decompose-task', {
  body: { taskTitle: topTask.title, spiciness: 5, locale },
});
const timeoutPromise = new Promise<never>((_, reject) =>
  setTimeout(() => reject(new Error('decompose-timeout')), 8_000)
);
const { data } = await Promise.race([invokePromise, timeoutPromise]);
```

### Figma MCP write tool

Use `mcp__Figma__use_figma` (or the longer-named MCP variant `mcp__3183acdc-cfa1-45b2-8690-39e117ec168e__use_figma`). Critical gotchas I hit this session:

1. **`figma.fileKey` returns `"headless"`** even when writing to a real file. Don't trust it to verify which file you're in. Use `mcp__Figma__whoami` and `create_new_file` returns to know the real fileKey.
2. **Font name for Inter is `"Semi Bold"` with a space**, not `"SemiBold"`. Same for `"Extra Bold"`.
3. **`figma.currentPage = page` is forbidden.** Use `await figma.setCurrentPageAsync(page)`.
4. **Auto-layout child stretching**: use `child.layoutSizingHorizontal = "FILL"` (newer API) instead of fiddling with `layoutAlign` + `primaryAxisSizingMode`.
5. **Variable color binding**: use `figma.variables.setBoundVariableForPaint(fill, 'color', variable)` to bind cleanly.
6. **Document name cannot be set via Plugin API.** `figma.root.name = "..."` throws. Use `create_new_file` to set the name at creation time.
7. **`setExplicitVariableModeForCollection(collection, modeId)`** on a frame applies the mode to that frame's entire subtree. This is how Energy-Low demonstration works.
8. **Cloned frames preserve children.** Use `findAllWithCriteria({types: ["TEXT"]})` to mass-edit text after clone.
9. **Effects API**: `node.effects = [{type: 'DROP_SHADOW', color: {r,g,b,a}, offset: {x,y}, radius, spread, visible, blendMode}]`. For glows: offset 0,0, color matches the element. For depth: offset 0,8, dark color, radius 24.
10. **Gradient paints**: `{type: 'GRADIENT_LINEAR' | 'GRADIENT_RADIAL' | 'GRADIENT_ANGULAR', gradientStops: [{position, color}], gradientTransform: [[a,b,c],[d,e,f]]}`. Variable binding on gradient stops is more complex than on solid fills — easier to hardcode hex values for accent gradients.
