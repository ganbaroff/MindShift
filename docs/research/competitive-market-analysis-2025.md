## 2. Конкурентная матрица — ключевые выводы

### Кто делает ADHD-дизайн хорошо
| Продукт | Что работает | Слабое место |
|---|---|---|
| **Structured** | Визуальные таймеры, цветовое кодирование, Apple Health | Нет Android AI, агрессивные попытки продать снова Lifetime-покупателям |
| **Tiimo** | Создан нейроотличными разработчиками, трекер настроения | Нет AI-декомпозиции, нет адаптации к перепадам энергии |
| **Routinery** | Алгоритмически ведёт шаг за шагом, снижает паралич | Агрессивный пейволл, баги с сохранением |
| **Goblin.tools** | Хирургически точный Magic ToDo, бесплатный | Полная изолированность, нет уведомлений, нет синхронизации |
| **Focusmate** | Клинически эффективный Body Doubling | Партнёры не приходят, нестабильность серверов |

### Кто делает плохо (и почему — урок для нас)
| Продукт | Ошибка | Следствие |
|---|---|---|
| **Motion** | Агрессивное авто-перепланирование 24/7 | «Движущаяся мишень» → когнитивный шок при СДВГ |
| **Notion** | Требует настройки системы перед использованием | Паралич исполнительной функции на онбординге |
| **Forest** | Убрал Lifetime в пользу подписки $60/yr, нет паузы таймера | Репутационный кризис, обрушение рейтингов |
| **Todoist/TickTick** | «Стена стыда» — просроченные задачи красным | Пользователь удаляет приложение, чтобы не видеть |

### Вакантная ниша (незаполнена)
> Между тяжёлыми корпоративными комбайнами (Motion, Notion) и изолированными утилитами (Goblin.tools, Forest) — **нет комплексного, психологически безопасного продукта**.

## 3. Незакрытые потребности — прямые выводы для MindFocus

### Критический Gap #1 — Guilt-Free Recovery
- Ни один конкурент не имеет встроенного протокола бесшовного возвращения
- Пользователь с СДВГ пропустил несколько дней → открывает приложение → красная стена просроченных задач → **удаляет приложение навсегда**
- **Нужно:** при возвращении — эмпатичный экран, автоматический архив старого в «Someday», один вопрос «Что важно сегодня?»

### Критический Gap #2 — Эпидемия контекстного переключения
- Средний удалённый сотрудник использует 9+ инструментов
- Типичный ADHD-стек: Google Calendar + Goblin.tools + Focusmate + Forest — **4 отдельных приложения**
- **Нужно:** единое пространство: оценка ресурса → декомпозиция → таймер → фокус

### Критический Gap #3 — AI как надсмотрщик vs AI как фильтр
- Motion = AI перепланирует всё хаотично → фоновая тревожность
- **Нужно:** AI как **пассивный умный фильтр**, скрывающий шум, а не агрессивный микроменеджер

## 4. Retention — реальные цифры рынка

| Метрика | Значение |
|---|---|
| D1 Retention (продуктивность) | 25–32.86% |
| D7 Retention | 18–24.23% |
| D30 Retention | 3.2–7.88% |
| Удаляют за первые 3 дня | до 77% DAU |
| Годовое удержание платящих (Android AI) | 11.5% медиана |

**Главная причина оттока D7+:** токсичное чувство вины при возвращении → решается Guilt-Free Recovery.

### Что доказанно повышает Retention

1. **Burnout Detection** — AI видит признаки перегрузки (HRV, хаотичный скроллинг) → упрощает UI до одного действия. Превращает приложение в «эмпатичного агента» → резко улучшает D7.

2. **Голосовой NLP Brain Dump** — поток сознания → AI структурирует, тегирует, декомпозирует. Вау-эффект в первые минуты → максимальный D1.

3. **Now / Next / Someday пулы** — показывать только 1-3 задачи в «Now». Всё остальное скрыто. Физиологически соответствует ADHD-восприятию времени («сейчас / не сейчас»).

## 5. Геймификация — что работает, что убивает

### Работает: мягкая RPG (Positive Reinforcement)
- **Finch** (питомец): не умирает, не наказывает за пропуски — просто **ждёт возвращения**
- Прогресс накапливается за выполненное, но **не обнуляется за невыполненное**
- Формирует устойчивую петлю привычки без стресса

### Убивает: Streak-системы и наказания
- Обнуление серии = спираль стыда = удаление приложения
- Красные счётчики = кортизол = избегание
- **Forest** — хрестоматийный пример репутационного краха через токсичный UX

### Текущая реализация в MindFocus
- ✅ Уже нет streaks, нет красных индикаторов (принцип ADHD-aware)
- ✅ XP-система (Bolt 4.1) — начислять за активность, не штрафовать за пропуск
- ⚠️ Нужно проверить Evening Review — нет ли там shame-inducing элементов

## 6. Монетизация — выводы для стратегии

### Ключевые цифры
| Метрика | Значение |
|---|---|
| CPI (iOS, AI-app) | > $10 |
| CPI (Android, Северная Америка) | ~$4 |
| Конверсия Free→Paid (дорогой тир) | 2.66% vs 1.49% |
| Конверсия Top 25% бизнес-приложений | 5.5% |
| LTV (iOS AI-app) | ~$37.80 |
| LTV (Web Billing) | до $255 |
| Оптимальный триал | 17–32 дня → конверсия 45.7% |
| 82% активаций триала | в день установки |

### Оптимальная модель 2025-2026: Hybrid
```
Базовый функционал (таймеры, оффлайн-календарь, локальная статистика)
→ Lifetime покупка или подписка $3-4/мес

Продвинутые AI-функции (NLP-парсинг, предиктивный анализ, аватары)
→ Пакеты расходуемых AI-токенов (Consumables)
```
- Закрывает боль аудитории с «subscription fatigue»
- Страхует от непредсказуемых API-расходов
- Совместима с рынком СНГ (нет рекуррентных платежей)

### Текущая модель MindFocus
- Free: 30 dumps/мес, 50 thoughts, 1 persona
- Pro: $8/мес — Unlimited

**Оценка:** базовая структура правильная, но стоит рассмотреть Lifetime опцию в пост-MVP для снижения subscription fatigue.

## 8. Технологические выводы

### Главный Gap конкурентов: виджеты и фоновые процессы
- Flutter конкуренты хорошо рендерят UI внутри приложения
- **Но:** плохо реализуют iOS/Android виджеты главного экрана
- Для ADHD-пользователя: «с глаз долой — из ума вон» (out of sight, out of mind)
- **Критическое требование:** запуск таймера и просмотр задачи с заблокированного экрана без входа в приложение

### Для MindFocus
- MindFocus — React PWA, не Flutter. Виджеты ОС недоступны для PWA нативно
- **Но:** PWA предлагает Web App Manifest shortcut actions — частичная компенсация
- Bolt 5.3 (Push Notifications) — шаг в сторону «нулевой дистанции» с заблокированного экрана

| **Delighters** (вау без ожидания) | Burnout Detection, Guilt-Free Recovery, async Body Doubling | ⚠️ Guilt-Free Recovery — добавить в возвращение пользователя |
