## 1. Нейробиология наград — выводы для нашего продукта

### Дофамин = предвкушение, не удовольствие
- Пик дофамина — в момент ТРИГГЕРА, не вознаграждения
- После получения награды уровень падает ниже базового → опустошение → поиск новой дозы
- **Вывод:** анимация перехода (открытие карточки, загрузка результата) важнее самой награды. Stretch the anticipation.

### Вариативное расписание наград (Variable Reward Schedule)
- Самое компульсивное расписание — непредсказуемое количество действий до награды
- Для СДВГ-мозга: **прямая дорога к поведенческой аддикции** (коморбидность СДВГ + IGD = до 83.3%)
- **Запрет для MindFocus:** никаких Gacha, Loot boxes, случайных наград

### Overjustification Effect (Эффект чрезмерного обоснования)
- Введение транзакционных внешних наград → уничтожает внутреннюю мотивацию
- «Я медитирую ради монет» → убрать монеты → человек перестаёт медитировать
- **Граница:** информационная обратная связь (прогресс-бар) — ОК. Контроль поведения (монеты за каждый чих) — опасно.
- **Для MindFocus:** XP = визуализация прогресса, не транзакционная валюта. Нет монет за задачи.

## 2. Streak-механики — точные данные

### Что работает (Duolingo-данные)
| Метрика | Значение |
|---|---|
| DAU/MAU Duolingo 2025 | 37.3% (медиана рынка: 22.4%) |
| Churn снижение 2020→2024 | 47% → 28% |
| 7-дневный streak → завершение курса | в 3.6x чаще |
| 7-дневный streak → D8 retention | в 2.4x чаще |
| >10 млн пользователей со streak | >365 дней |

### Что ломается
- После круглой цели (100 дней) — **резкое падение мотивации**
- Потеря streak = **личный провал** → abandonment приложения
- Streak Repair за деньги → **обесценивает усилия** → парадокс снижения мотивации

### Для СДВГ: streak категорически противопоказан в жёсткой форме
- Наказывает за естественные флуктуации энергии
- Срыв → спираль стыда → удаление приложения

### Текущее состояние MindFocus
- ✅ Streaks убраны — правильное решение
- ✅ XP начисляется за активность, не убывает
- ⚠️ Нужна визуализация «мягкого» роста — не счётчик дней, а резервуар/растение

## 4. Эффект Протея — ключевой инсайт для Persona-системы

**Суть (Yee & Bailenson, 2007):** пользователи бессознательно перенимают поведение и черты характера своего аватара.
- Высокий аватар → переговоры агрессивнее
- Привлекательный аватар → больше общительности

**Для MindFocus Persona (Bolt 3.1-3.2):**
- Пользователь настраивает архетип (Explorer/Builder/Dreamer/Guardian)
- Архетип транслирует определённые качества → **прайминг реального поведения**
- Человек начинает вести себя в соответствии со стандартами своего цифрового альтер-эго
- **Это усиливает ценность Persona-системы** — она не просто косметика, это поведенческий инструмент
- **Развитие:** позволить аватару «выглядеть» компетентным, собранным → пользователь следует за образом

## 5. Кейсы — конкретные уроки

### Finch — ближайший аналог нашей Persona-системы
- 5 млн MAU, D30 = **22%** (феноменально для ниши)
- Механика: забота о птице = забота о себе (метафора совпадает с целью)
- Птица **никогда не умирает**, не наказывает за отсутствие
- При возвращении после недели: **встречает с радостью**, не с упрёком
- Поощряет минимальные микро-действия (выпить воды)
- **Вывод:** наша Persona должна работать так же — не умирает, ждёт, радуется возвращению

### SuperBetter — парадокс терапевтической эффективности
- Клинически доказано: снижение депрессии (Cohen's d=0.67) и тревожности за 6 недель
- Провалился коммерчески: пользователи выздоравливали и уходили
- **Вывод:** не делать продукт, который нужен только в кризисе. MindFocus должен быть полезен и в «нормальные» дни.

### Habitica — урок о масс-маркете
- Работает для геймеров, провально для обычных пользователей
- HP Loss за невыполнение → **фатально для СДВГ/депрессии**
- Высокий порог входа (статы, экипировка, мана) → когнитивная перегрузка
- **Вывод:** RPG-глубина — не для всех. MindFocus должен быть RPG-лайт без видимых механик

### Duolingo — стресс vs метрики
- Лиги и страйки: DAU/MAU 37.3%, но **пользователи фармят простые уроки ради рейтинга**
- Признают тревожность, выгорание от лиг
- **Вывод:** метрики растут, но внутренняя мотивация подменяется внешней — это долгосрочная угроза

## 6. Self-Determination Theory — три потребности для MindFocus

| Потребность | Что нужно реализовать |
|---|---|
| **Автономия** | Пользователь сам выбирает квесты/задачи. Нет принудительных dailies. |
| **Компетентность** | Прозрачная визуализация роста навыков. Адаптивная сложность (баланс = состояние потока). |
| **Причастность** | Асинхронные кооперативные механики. Нет соревновательности. Есть совместное созидание. |

## 8. Три механики к внедрению (из отчёта)

### 1. AI Body Doubling — компаньон во время фокус-сессии
- Аватар «работает рядом» во время таймера
- Раз в 15 минут — короткое текстовое одобрение (AI)
- **Прямая связь с Bolt 5.2** — Focus Mode + Persona уже есть → добавить «присутствие» во время таймера
- Scope: 1-2 недели поверх Bolt 5.2

### 2. Гибкий streak — резервуар/растение (Forgiving Streak)
- Не счётчик дней, а убывающий/растущий резервуар
- Пропуск → слегка понижает, не обнуляет
- **Проверить:** наша XP-система уже не штрафует, но нет визуализации «мягкого роста»
- Scope: 1 неделя — модификация XP-индикатора

### 3. Косметическая кастомизация за задачи (не Gacha)
- Стабильная (не случайная!) валюта за задачи → одежда/фоны аватара
- Не влияет на эффективность, не за реальные деньги
- Активирует Эффект Протея без компульсивности
- **Связь:** PersonaCard (Bolt 3.1) — готовая точка расширения
- Scope: 2 недели (система инвентаря + спрайты)

### Три запрещённые механики
1. ❌ HP Loss / убывающее здоровье за пропуски
2. ❌ Gacha / Loot boxes / случайные награды
3. ❌ Публичные Leaderboards
